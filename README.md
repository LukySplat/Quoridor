<h4 align="center">In-game commands.</h4>
<h5>- s: Move forward (Player 1) / Move back (Player 2)</h5>
- z: Move back (Player 1) / Move forward (Player 2)
- d: Right
- q: Left
- w: Walls
- u: Save game
- p: Pause
